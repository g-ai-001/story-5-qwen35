# 末日游戏降临：无限流副本中的救赎

## 简介

末日降临，现实世界与"深渊游戏"融合，人类必须在各种恐怖副本中求生。重生者顾林带着前世记忆回归，却发现自己只有三个月寿命。为了活下去，他必须进入一个个包含东方神话、克苏鲁、赛博朋克风格的副本，通过暴打"魔主"爆装备，挑战系统规则，揭开末日背后的真相。

## 类型

无限流/末世小说

## 看点

融合辰东的"逆天"与番茄的"闯关"，节奏极快，世界观多元
