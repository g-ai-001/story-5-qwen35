# 第三十一章：血族公爵

## 第三十一章：血族公爵

顾林跟着四大领主穿过城堡的长廊。

长廊两侧挂满了血红色的挂毯，上面绣着复杂的图案——有展翅的蝙蝠，有盛开的血色玫瑰，还有各种各样的血族纹章。

空气中弥漫着一股陈旧的血腥味，让人感到一阵压抑。

"前面就是血族公爵的领地。"德古拉的声音从前方传来，"他叫做马库斯，是该隐大人最初的后裔之一。"

"最初的后裔？"顾林问道。

"没错。"德古拉的眼中闪过一丝复杂的情绪，"在血族的历史上，只有极少数的存在能够获得该隐大人的初拥。而马库斯……就是其中之一。"

"那他为什么要背叛该隐？"

"因为权力。"德古拉冷笑，"马库斯一直认为，他比该隐大人更适合统治血族。所以他一直在暗中扩张自己的势力，直到有一天……"

他顿了顿，继续说道："直到有一天，他遇到了深渊之子。"

"深渊之子？"顾林的瞳孔猛地收缩。

"没错。"德古拉点头，"深渊之子向马库斯许诺，只要他能够帮助深渊之子取代该隐大人，他就能成为新的血族之主。"

"马库斯接受了这个提议？"

"是的。"德古拉的眼中闪过一丝杀意，"所以，他成为了血族的背叛者。"

顾林沉默了。

他突然意识到，这一切都比他想象的要复杂得多。

深渊之子、血族背叛者、还有三个月后的浩劫……

这一切都在将他推向一个巨大的漩涡。

"到了。"德古拉的声音打断了他的思绪。

顾林抬起头，发现自己已经来到了一座巨大的铁门前。

铁门上雕刻着一个巨大的蝙蝠图案，蝙蝠的眼睛是两颗血红色的宝石，在黑暗中闪烁着诡异的光芒。

"这里是马库斯的私人领地。"德古拉说道，"我们只能送你到这里了。"

"为什么？"

"因为这是血族的规矩。"德古拉的眼中闪过一丝无奈，"血族公爵的领地，只有血族认可的人才能进入。"

"而我们……"他苦笑了一下，"并不被马库斯认可。"

顾林皱眉。

"那我怎么进去？"

"你？哈哈……"德古拉笑了起来，"你不需要进去。因为……"

他的话还没说完，铁门突然缓缓打开。

一个身穿黑色长袍的男子从门内走出。

他的面容苍白而英俊，嘴角挂着一抹似笑非笑的弧度，一双猩红色的眼睛在黑暗中闪烁着危险的光芒。

"欢迎来到我的领地。"男子的声音低沉而沙哑，"顾林先生。"

"你是……"顾林的手已经握住了冥府·克图格亚。

"我是马库斯。"男子微微鞠躬，"血族公爵，马库斯。"

"欢迎你的到来。"

顾林看着他，心中涌起一股警惕。

他能够感觉到，这个男人的实力……非常强大。

甚至比四大领主加起来都要强。

"你早就知道我会来？"顾林沉声问道。

"当然。"马库斯的嘴角勾起一抹笑容，"该隐的每一步棋，我都知道。"

"那你为什么还要让我进来？"

"因为我想见见你。"马库斯的眼中闪过一丝赞赏，"传说中的重生者，顾林。"

"你是怎么知道的？"

"这个世界上，没有我不知道的事情。"马库斯张开双臂，"比如说……我知道你只剩45天寿命了。"

顾林的瞳孔猛地收缩。

"我知道你在寻找延长寿命的方法。"马库斯继续说道，"我也知道，该隐答应给你'血族之心'，作为杀死我的报酬。"

"但是……"他的笑容变得更加深邃，"你有没有想过，该隐为什么要让你来杀我？"

"为什么？"

"因为我掌握着一个秘密。"马库斯的眼中闪过一丝光芒，"一个关于深渊游戏真相的秘密。"

"如果你杀了我……你就永远也不会知道这个秘密了。"

顾林沉默了。

他看着马库斯，心中涌起一股复杂的情绪。

"你想说什么？"他问道。

"我想说……"马库斯朝他伸出手，"跟我合作。"

"什么？"

"跟我合作，而不是跟该隐合作。"马库斯的眼中闪烁着真诚的光芒，"我可以告诉你深渊游戏的真相。我可以给你真正的永生。而你……只需要帮我杀死该隐。"

顾林看着他，沉默了片刻。

然后，他笑了。

"你觉得我会相信你吗？"他问道。

"为什么不相信？"马库斯反问。

"因为……"顾林的眼中闪过一丝寒光，"深渊之子是我最大的敌人。"

"而你，是深渊之子的盟友。"

马库斯的眼中闪过一丝失望。

"那真是太可惜了。"他叹了口气，"看来……我们只能用另一种方式解决问题了。"

他的身后，突然涌出了无数道血红色的身影。

那些都是血族的战士——数量至少有上百个。

"你以为我会一个人来？"马库斯冷笑，"顾林，你太天真了。"

顾林看着那些血族战士，脸上没有丝毫惧色。

"我从来没说我是一个人来的。"

马库斯的瞳孔猛地收缩。

"什么意思？"

顾林没有回答，只是抬起了手。

一道光芒闪过。

白嫖怪、晴天、小财神、沈碧——四个人同时出现在他身后。

"老大，我们来了！"白嫖怪举起长刀，眼中闪烁着战意。

"哼。"沈碧冷哼一声，长剑出鞘，"我说过，我不会让你一个人战斗。"

马库斯的脸色变得阴沉。

"你们……"

"很意外吗？"顾林的嘴角勾起一抹笑容，"我说过，我从不打没有准备的仗。"

"所以……"

他举起冥府·克图格亚，刀身爆发出幽蓝色的光芒。

"让我们开始吧。"
