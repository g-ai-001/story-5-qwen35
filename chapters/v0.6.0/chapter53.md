# 第五十三章：另一个时间线

## 第五十三章：另一个时间线

沈碧走进大厅，看到一个年轻人站在窗边。

他看起来二十多岁，身材高大，五官俊朗。

但最引人注目的是他的眼睛——一双深邃的、仿佛能看穿一切的眼睛。

"你就是顾辰？"沈碧问道。

年轻人转过身，对她微微鞠躬。

"是的，阿姨。"

"我是顾辰。"

"来自另一个时间线。"

沈碧仔细打量着他。

他的眉眼之间，确实有顾林的影子。

但又有些不同。

顾林的眼神总是带着一丝忧郁，而他的眼神却更加沉稳。

"你说你是顾林的儿子？"沈碧问道，"可是据我所知，顾林只有一个女儿。"

"就是我收养的念念。"

顾辰沉默了片刻。

"阿姨，我来自的时间线，和您所在的时间线不同。"

"在那个时间线里，父亲没有封印深渊之心。"

"他选择了另一条路。"

沈碧的眉头微皱。

"什么路？"

"他选择和深渊之子同归于尽。"顾辰的声音低沉。

"那一战，父亲赢了。"

"但他也付出了生命的代价。"

"而我……"

他顿了顿。

"我是父亲和深渊之子战斗时，深渊之心残留的力量与他体内因子结合后诞生的存在。"

"我继承了父亲的一部分记忆和力量。"

"也继承了深渊之子的一部分力量。"

沈碧的脸色微变。

"你是……深渊的后裔？"

"不完全是。"顾辰摇了摇头。

"我体内确实有深渊的力量，但更多的是父亲的因子。"

"父亲在临死前，用自己的力量净化了我体内的深渊力量。"

"所以我不会像石磊那样被深渊吞噬。"

"我存在的意义，是完成父亲未完成的事业。"

沈碧看着他，心中涌起复杂的情感。

这个年轻人，是顾林的另一个儿子。

虽然不是她亲生的，但顾林的血脉。

"你来这里做什么？"她问道。

"我想见见您。"顾辰说道。

"也想看看，父亲用生命换来的这个世界。"

他的目光变得柔和。

"还有……我的妹妹。"

沈碧愣了一下。

"妹妹？"

"是的。"顾辰点头。

"在父亲前世的记忆里，他一直很遗憾没能见到我。"

"现在，父亲在这个时间线有了女儿。"

"对我而言，她就是我的妹妹。"

沈碧沉默了。

她不知道该怎么面对这个情况。

这个突然出现的"儿子"，让她有些手足无措。

"沈碧。"

白嫖怪的声音从门口传来。

"这个人来历不明，不能轻易相信。"

他的眼神警惕地看着顾辰。

"万一是深渊的残余势力假扮的呢？"

"我不是。"顾辰平静地说。

"我可以证明。"

他抬起手，一道光芒从他掌心浮现。

那是一道金色的光芒，温暖而柔和。

"这是父亲的力量。"他说，"净化之力。"

"只有真正继承父亲因子的人，才能使用这种力量。"

白嫖怪的脸色微变。

他见过这种力量。

在林远身上。

"如果你还不相信。"顾辰继续说道。

"我可以接受任何测试。"

沈碧看着顾辰的眼睛。

那双眼睛里，没有欺骗，没有恶意。

只有真诚和思念。

"我相信你。"她说道。

"但是……"

她看着顾辰。

"你确定要留下来吗？"

"这里不是你的时间线。"

"这里没有你的父亲。"

顾辰沉默了。

"我知道。"他说。

"但父亲用生命换来的这个世界，我想亲眼看看。"

"想看看他守护的东西。"

"想看看……他爱的人。"

他看向沈碧。

"包括您，阿姨。"

"我知道父亲很爱您。"

"在他的记忆里，您是他最重要的人。"

沈碧的眼眶湿润了。

"好孩子……"她喃喃道。

"你想留下来，就留下来吧。"

"这里以后就是你的家。"

顾辰的眼中闪过一丝感动。

"谢谢您，阿姨。"

他深深地鞠了一躬。

"谢谢您愿意接纳我。"
