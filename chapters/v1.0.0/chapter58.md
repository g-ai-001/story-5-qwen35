# 第五十八章：故人重逢

## 第五十八章：故人重逢

三个月后。

顾辰已经成为了深渊管理局最出色的战斗员。

他参与了数十次清剿深渊残余势力的行动，立下了赫赫战功。

但他心里清楚，这些都只是小打小闹。

真正的敌人——深渊八歧——从未现身。

这一天下午。

顾辰正在基地的训练室里修炼。

突然，警报声响起。

"警告，警告，有不明人员闯入基地！"

"重复，有不明人员闯入基地！"

顾辰猛地睁开眼睛。

他冲出训练室，朝警报响起的地方跑去。

基地的大厅里，一片混乱。

十几个士兵围成一个圈，将一个身影包围在中央。

那个身影穿着一身破旧的黑色风衣，脸上有一道狰狞的伤疤，从额头一直延伸到下巴。

他的身上散发着强大的气势，压得周围的士兵们几乎喘不过气来。

"都退下！"

顾辰大喊一声。

士兵们如蒙大赦，纷纷后退。

顾辰走上前去，直视着那个神秘人物。

"你是谁？"

那个人抬起头，看向顾辰。

他的眼神中闪过一丝复杂的情绪。

"你是……顾林的儿子？"

顾辰的眼神一凝。

"你认识我父亲？"

那个人沉默了片刻。

然后，他笑了。

"岂止是认识。"

他伸出手，在脸上轻轻一抹。

那道狰狞的伤疤竟然开始消退，露出下面的真容。

那是一个中年男子，面容刚毅，眼神深邃。

顾辰认得他。

不，准确地说，他在顾林留下的记忆碎片中见过他。

"你是……白嫖怪叔叔？"

那个人愣了一下。

然后，他哈哈大笑起来。

"没想到顾林那小子，连这个都告诉你了。"

"没错，我就是白嫖怪。"

"只不过……"他的笑容渐渐收敛，"现在大家都叫我'暗夜'。"

---

会客厅里。

顾辰和白嫖怪相对而坐。

"这三年，你都去了哪里？"顾辰问道。

白嫖怪沉默了一会儿。

"三年前，深渊游戏结束后，我们都以为一切都结束了。"

"但我隐约感觉到，深渊的威胁并没有完全消除。"

"于是我开始暗中调查。"

"结果发现，有一股更加黑暗的势力在暗中活动。"

"那股势力的首领，就是深渊八歧。"

顾辰的眼神变得凝重。

"你知道深渊八歧？"

"当然知道。"白嫖怪点头，"他曾经是深渊游戏的创建者之一。"

"在深渊之子被封印后，他带着一部分深渊残余势力逃了出来。"

"这三年来，他一直在暗中招兵买马，积蓄力量。"

"现在，他的势力已经非常庞大了。"

"你来找我，是为了什么？"

白嫖怪看着顾辰，眼神变得认真起来。

"我来找你，是为了警告你。"

"深渊八歧对你非常感兴趣。"

"他知道你体内有打开深渊之门的钥匙。"

"他一直在寻找机会，想要夺取那把钥匙。"

顾辰点头。

"这个我知道。"

"不，你不知道。"白嫖怪摇头，"你不知道的是，深渊八歧已经找到了打开深渊之门的另一种方法。"

"不需要钥匙，也能打开深渊之门。"

顾辰的脸色大变。

"什么？！"

"我调查了很久，终于发现了这个秘密。"白嫖怪的声音变得低沉。

"深渊之门的本质，是连接深渊和现实世界的通道。"

"这把钥匙的作用，是稳定这个通道。"

"但如果用另一种方式……用大量的生命作为祭品，也能强行打开通道。"

"只不过那种方式打开的通道非常不稳定，只能维持很短的时间。"

"但对深渊八歧来说，这就足够了。"

"只要能打开通道哪怕一秒钟，他就能让深渊之主的一小部分意识降临这个世界。"

"而一旦深渊之主的意识降临，哪怕只是一小部分……"

"这个世界，都将面临灭顶之灾。"

顾辰的拳头握紧。

"所以我必须阻止他。"

"没错。"白嫖怪点头，"但这非常困难。"

"深渊八歧的实力，已经超过了Lv.100。"

"在这个世界上，能够与他抗衡的人屈指可数。"

"而你……"他看着顾辰，"虽然潜力巨大，但现在还远远不是他的对手。"

"我需要变得更强。"顾辰说道。

"是的。"白嫖怪点头，"但在那之前，你需要找到一样东西。"

"什么东西？"

"顾林当年留下的遗物——净化之核。"

"那是顾林凝聚毕生功力创造出的神器，蕴含着能够彻底消灭深渊的力量。"

"如果你能得到它，或许就能与深渊八歧一战。"

顾辰的眼中闪过一丝希望的光芒。

"净化之核……在哪里？"

"就在你身边。"白嫖怪微微一笑。

"它一直藏在希望之家孤儿院的地下。"

"沈碧一直守护着它。"

"只是她不知道，那件东西真正的用途。"

顾辰的心跳加速。

原来，母亲一直守护着的，不仅仅是父亲留下的记忆。

还有能够彻底消灭深渊的力量。
